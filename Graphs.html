ABV Distribution




Figure 4. The average ABV for this dataset is around 8%. However, there are a large number of outliers both at the lower and higher end of the ABV range.



Logistic Regression Results




Figure 5. The above is the receiver operating characteristic curve (ROC) that plots the true positive rate (TPR) against the false positive rate (FPR). TPR represents the probability of detection. The FPR represents the probability of a false alarm. The diagonal line represents where points would fall in the model if it were randomly guessing; therefore, not fitting the data well. Points above the diagonal line represent good classification results from the logistic regression model.




Figure 6. Precision represents the ration of the number of true positives to the sum of the numbers of true and false positives. Recall represents the sensitivity of the model and is the ratio of the number of true positives to the number of actual positives. The highest possible f-score is 1.0 which indicates perfect precision and recall. Support represents the number of each category in y-test.
